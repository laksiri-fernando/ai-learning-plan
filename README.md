# **My Plan for Learning AI**

# **🚀 10-Week Applied AI Plan (Zero → Production-Ready)**

## **🎯 End Goal**

By Week 10, you will have:

* A **working AI-powered feature** (e.g., material prediction / AI assistant)

* A **deployed API on Kubernetes**

* A **clear AI roadmap for your product**

---

# **🧱 WEEK 1–2: Python \+ Data Fundamentals (Fast Track)**

### **🎯 Goal**

Get just enough Python \+ data skills (no over-learning)

### **Learn:**

* Python basics (focus only on usage, not theory)

* NumPy

* Pandas

### **Do:**

* Load CSV (your real production data if possible)

* Clean \+ transform data

* Aggregate business metrics

### **Mini Project:**

* “Material usage summary dashboard dataset”

👉 Output:

* Clean dataset ready for ML

---

# **🤖 WEEK 3–4: Core Machine Learning (Business Focus)**

### **🎯 Goal**

Build your first **real prediction model**

### **Learn:**

* scikit-learn

* Regression (predict numbers)

* Classification (predict categories)

* Train/test split

### **Do:**

Build 1 model:

* Example: **Material availability prediction**

* Or: delivery delay prediction

### **Mini Project:**

* Input: historical data

* Output: predicted value (e.g., shortage risk)

👉 Output:

* `.pkl` trained model file

---

# **🧠 WEEK 5–6: GenAI (HIGH IMPACT for you)**

### **🎯 Goal**

Add **AI intelligence layer to your product**

### **Learn:**

* Large Language Models

* Prompt engineering

* RAG (Retrieval-Augmented Generation)

### **Tools:**

* OpenAI API or open-source models

* LangChain

### **Build:**

👉 “AI Assistant for KingslakeBlue”

* Ask: *“Why is line 3 delayed?”*

* AI answers using your data

### **Mini Project:**

* Connect LLM \+ your database (RAG)

👉 Output:

* Working chatbot for internal use

# **⚙️ WEEK 7: Model Optimization \+ Evaluation**

### **🎯 Goal**

Make your model usable in production

### **Learn:**

* Accuracy metrics (MAE, RMSE)

* Feature importance

* Basic tuning

### **Improve:**

* Clean bad data

* Remove useless features

👉 Output:

* Better-performing model
